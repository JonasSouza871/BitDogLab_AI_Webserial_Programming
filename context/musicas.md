# Banco de Musicas para Buzzer (GPIO 21)

Use o formato (frequencia, duracao) para tocar. R=0 é silencio/pausa.
SEMPRE use este codigo base:
```python
from machine import Pin, PWM
import utime
bz = PWM(Pin(21))
for freq, dur in musica:
    if freq == 0:
        bz.duty_u16(0)
    else:
        bz.freq(freq); bz.duty_u16(32768)
    utime.sleep(dur); bz.duty_u16(0); utime.sleep(0.05)
bz.deinit()
```
